ilutEnable, ilutDisable

These functions enable or disable an ILUT state.

 

ILboolean ilutEnable(
  ILenum Mode
);


ILboolean ilutDisable(
  ILenum Mode
);

Parameters

Mode
Mode to enable/disable.

Remarks

ilutEnable enables a mode specified by Mode. ilutDisable disables a mode specified by Mode.

A list of available modes and their meaning:

ILUT_PALETTE_MODE
Does nothing currently.

ILUT_OPENGL_CONV
When enabled, DevIL passes GL_RGB8 and GL_RGBA8 to OpenGL instead of GL_RGB and GL_RGBA. This can give better visual quality on some nVidia-based cards (at the price of increased memory usage).

Error Codes

ILUT_INVALID_ENUM - Mode was of an unsupported value.
