ilutGLBuildMipmaps

The ilutGLBuildMipmaps function creates mipmaps from an image for OpenGL to use.

 

ILboolean ilutGLBuildMipmaps(
  ILvoid
);

Parameters

None

Remarks

ilutGLBuildMipmaps generates mipmaps via gluBuild2DMipmaps from an image. This function is similar to ilutGLTexImage but creates mipmaps.

Error Codes

ILUT_ILLEGAL_OPERATION - There is currently no image bound. Use ilGenImages and ilBindImage before calling this function.

See Also

ilGenImages, ilBindImage, ilutGLBindMipmaps
