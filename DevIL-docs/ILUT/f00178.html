ilutPopAttrib

The ilutPopAttrib function "pops" the last entry off the state stack into the current states.

ILvoid ilutPopAttrib(
  ILvoid
);

Parameters

None

Remarks

ilutPopAttrib pops the last pushed stack entry off the stack and copies the bits specified when pushed by ilutPushAttrib to the previous set of states.

Error Codes

ILUT_STACK_UNDERFLOW - The stack was popped without being pushed by ilutPushAttrib.
