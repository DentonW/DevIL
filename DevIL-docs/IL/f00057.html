ilInit

The ilInit function initializes DevIL.

ILvoid ilInit(
  ILvoid
);

Parameters

None

Remarks

ilInit starts DevIL and must be called prior to using DevIL, or else DevIL will probably crash when you attempt to use it.

Error Codes

None

See Also

iluInit, ilutInit, ilShutDown
