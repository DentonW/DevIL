IL_ORIGIN_SET

 

L Mode Value. When enabled, the origin is specified at an absolute position, and all images loaded or saved adhere to this set origin. For more information, check out ilOriginFunc.


Usage

 
ilDisable, ilEnable, ilGetBoolean, ilGetBooleanv, ilGetInteger, ilGetIntegerv, ilIsDisabled, ilIsEnabled,
