ilRemoveSave

The ilRemoveSave function removes a registered saving function from OpenIL's registered functions list.

 

ILboolean ilRemoveSave(
  char *Ext
);

Parameters

Ext
Extension to remove (e.g. "tga").

Remarks

ilRemoveSave removes a registered extension handler from the registered save functions list. Use this function when a new handler for an extension needs to be registered.

Error Codes

None

See Also

ilRegisterSave, ilRemoveLoad, ilRegisterLoad
